## 🚀 Overview

This project focuses on predicting the **popularity of Facebook events** using machine learning techniques. The model analyzes key event attributes to determine their likelihood of success, helping event organizers anticipate interest and plan effectively.

---

## 🌟 Key Features

- 📋 Preprocessing of Facebook event datasets
- 📊 Visualizations of user interests & invite patterns
- 🧠 Machine Learning model trained on labeled data
- 🌐 Flask-based web interface for real-time interaction
- 📈 Graphical display of prediction outputs

---

| Category         | Tools / Libraries                  |

| Web Framework    | Flask                              |
| Data Handling    | Pandas, NumPy                      |
| Visualization    | Matplotlib, Seaborn, Plotly        |
| UI               | HTML, CSS (via Flask templates)    |

---

## 🧪 Workflow

1. Load and preprocess event data.
2. Analyze engagement metrics.
3. Train a machine learning classifier (e.g., Random Forest).
4. Build a Flask-based web app to predict outcomes.
5. Display results and charts dynamically.

## 🎯 Results

- Achieved accuracy: **87–92%** depending on model
- Evaluation metrics: Confusion matrix, precision, recall, F1-score
- Effective at predicting 'high interest' vs 'low interest' classes
